# Miscellaneous MCP Servers

This directory contains external and experimental MCP servers.

## Purpose

- External MCP servers cloned from GitHub
- Experimental tools under development
- Third-party integrations

## Structure

Each subdirectory should contain:
- MCP server implementation
- README with setup instructions
- Configuration files

## Management

Use `scripts/sync_github_mcps.py` to clone and update external repositories.
